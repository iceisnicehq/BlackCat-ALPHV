BlackCat, also known as ALPHV or Noberus, is a ransomware family and cybercrime group operating on a Ransomware-as-a-Service (RaaS) model. It first appeared in November 2021, written in the Rust programming language, and targeted hundreds of organizations worldwide, including high-profile victims like Reddit and Change Healthcare.
